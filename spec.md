## STEP 1: 入力画面

### 生年月日入力

- 年：1930〜2010のセレクトボックス
- 月：1〜12のセレクトボックス
- 日：1〜31のセレクトボックス
- 算命学では立春（2月4日頃）を年の区切りとして扱う

### 手相チェックリスト（写真を見ながら選択）

| 線 | 選択肢 |
|----|--------|
| 生命線 | 長い / 普通 / 短い / 枝分かれ |
| 感情線 | 長い / 短い / カーブが強い / まっすぐ |
| 知能線 | 長い / 短い / カーブ / 二股 |
| 運命線 | ある / 薄い / ない |
| 太陽線 | ある / ない |

### MBTIタイプ（任意）

- 16タイプのドロップダウン（未選択可）
- INTJ / INTP / ENTJ / ENTP / INFJ / INFP / ENFJ / ENFP
- ISTJ / ISFJ / ESTJ / ESFJ / ISTP / ISFP / ESTP / ESFP

## STEP 2: 計算ロジック

### 2-1. 算命学（`lib/sanmeigaku.ts`）

**概要：** 中国由来の占術。生年月日から「天干地支」を算出し、人生の傾向や性格を読む。

**計算フロー：**

```
生年月日 → 立春補正 → 年・月・日の天干地支 → 日干を軸に主星を特定 → 解釈テキスト出力
```

**天干（10種）：** 甲・乙・丙・丁・戊・己・庚・辛・壬・癸
**地支（12種）：** 子・丑・寅・卯・辰・巳・午・未・申・酉・戌・亥
**干支60周期（六十干支）で年・月・日を表現**

**立春補正：** 2月4日以前の生まれは前年の干支を使用

**十大主星（日干と各柱の蔵干の関係で決定）：**

| 主星 | キーワード |
|------|------------|
| 天将星 | 帝王・リーダー・激しさ |
| 天禄星 | 安定・現実主義・誠実 |
| 天権星 | 知性・論理・分析 |
| 天奉星 | 奉仕・献身・優しさ |
| 天庫星 | 蓄積・忍耐・粘り強さ |
| 天極星 | 変革・革新・自由 |
| 天南星 | 情熱・行動力・直感 |
| 天胡星 | 創造・芸術・感受性 |
| 天報星 | 縁・運・人気 |
| 天貴星 | 品格・高貴・精神性 |

**出力：**
- 年干支・日干
- 主星名
- 性格傾向（テキスト）
- 才能（テキスト）
- 対人関係パターン（テキスト）

---

### 2-2. 5アニマル占い（`lib/fiveAnimals.ts`）

**概要：** 生年月日から命数（1〜60）を算出し、5種の動物（各リーダー/サブ計10タイプ）に分類する。

**計算フロー：**

```
生年月日 → 年月テーブルからベース数を取得 → + 日付 → 60を超えたら-60 → 命数1-60
→ 動物タイプテーブルで分類 → 解釈テキスト出力
```

**10タイプ：**

| 動物 | リーダー | サブ |
|------|---------|------|
| 虎 | 虎(L) | 虎(S) |
| 狼 | 狼(L) | 狼(S) |
| 猿 | 猿(L) | 猿(S) |
| コアラ | コアラ(L) | コアラ(S) |
| 黒ヒョウ | 黒ヒョウ(L) | 黒ヒョウ(S) |

**出力：**
- 命数
- 動物名・タイプ（L/S）
- 性格（テキスト）
- 恋愛スタイル（テキスト）
- 仕事スタイル（テキスト）

---

### 2-3. 手相占い（`lib/palmReading.ts`）

**概要：** 5線のチェックリスト入力から性格キーワードを抽出し、総合傾向を分析する。

**各線の意味：**

| 線 | 読み解き観点 |
|----|-------------|
| 生命線 | 生命力・健康・行動力 |
| 感情線 | 感情表現・愛情・対人感受性 |
| 知能線 | 思考傾向・知性・判断力 |
| 運命線 | 社会運・目標意識・努力 |
| 太陽線 | 人気・才能・幸運 |

**出力：**
- 性格キーワードリスト（3〜5個）
- 総合サマリー（テキスト）

---

### 2-4. MBTIデータ（`lib/mbti.ts`）

**概要：** 選択されたMBTIタイプに対応する特性テキストを返す。

**出力：**
- タイプ名
- 性格説明
- 恋愛スタイル
- 仕事スタイル

---

### 2-5. 統合ロジック（`lib/integrate.ts`）

**概要：** 各占い結果を組み合わせて「統合人物像」を生成する。

**ロジック：**
- 各占いの性格キーワードを集約
- 重複・強調されるキーワードを軸に統合人物像を構築
- 恋愛タイプ・仕事タイプ・キャッチコピーを生成

**出力：**
- 統合人物像（3〜5文）
- 恋愛タイプ（例：「リードしたいが束縛はしない自由恋愛型」）
- 仕事タイプ（例：「縁の下の力持ち型・専門職向き」）
- 一言キャッチコピー（例：「静かなる革命家」）

## STEP 3: 結果画面

### 表示順序（アニメーションで順次フェードイン）

1. ローディング演出（神秘的なアニメーション）
2. 算命学カード（主星・天干地支・性格傾向）
3. 5アニマルカード（動物タイプ・特性）
4. 手相カード（各線の読み解き・キーワード）
5. MBTIカード（タイプ特性）※入力がある場合のみ
6. **統合人物像**（3〜5文・最もドラマチックに表示）
7. 恋愛タイプ
8. 仕事タイプ
9. キャッチコピー（最大フォントで表示）

### SNSシェア機能

- 「結果をコピー」ボタン
- クリップボードに以下のテキスト形式でコピー：

```
【占い統合診断 結果】
🌟 キャッチコピー：{catchphrase}

⭐ 算命学：{mainStar}
🐾 5アニマル：{animal}タイプ
💫 統合人物像：{characterDescription}

💕 恋愛タイプ：{loveType}
💼 仕事タイプ：{workType}

#占い統合診断 #性格診断
```

- **見出し：** Noto Serif JP Bold（神秘的）

- **キャッチコピー：** 特大フォント（32px+）・金色グラデーション

- モバイルファースト

- カード間隔：16px

## 非機能要件

- **パフォーマンス：** 全計算はクライアントサイドで完結（API不要）
- **プライバシー：** 入力データはブラウザ外に送信しない
- **アクセシビリティ：** 適切なaria-label・セマンティックHTML
- **SEO：** Open Graph メタタグ設定（結果シェア用）
